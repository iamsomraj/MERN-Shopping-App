# MERN-Shopping-App

MERN-Shopping-App is a repository, which contains an app, built using MongoDB, Express, React and Nodejs. The main state of the application is maintained using Redux.

## Features

1. Products
2. Pagination
3. Orders
4. Carts
5. Payment Gateway
6. Users
7. Admin
